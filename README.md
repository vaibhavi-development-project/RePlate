# 🍱 RePlate
## Food Waste Reduction Platform

A modern frontend-focused food waste reduction platform developed to help restaurants reduce surplus food wastage by connecting them with nearby users through discounted food listings, QR-based reservations, and responsive user-focused design.

React • Next.js • Tailwind CSS • Node.js

# 🚀 Project Overview

RePlate is a food waste reduction platform designed to help restaurants showcase surplus meals and allow nearby users to discover affordable food options through a clean and responsive interface.

The project mainly focused on:

- Food wastage awareness
- Responsive frontend architecture
- QR-based reservation flow
- Role-based UI structure
- Smooth user interaction
- Real-time-like admin synchronization

# ⚙️ Installation & Setup

## Clone Repository

```bash
git clone https://github.com/vaibhavi-development-project/replate.git
```

## Navigate to Project

```bash
cd replate
```

## Install Dependencies

```bash
npm install
```

## Run Development Server

```bash
npm run dev
```
